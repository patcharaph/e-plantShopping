# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React-based shopping cart application for an online plant store called **Paradise Nursery**.  
This application allows users to browse different categories of house plants, add plants to a shopping cart, update quantities, and view the total cost dynamically.

### Features
- Landing page with navigation to products and cart
- Product listing with plant images, descriptions, and prices
- Add to Cart functionality using Redux
- Shopping cart with quantity update and remove options
- Real-time cart quantity and total price updates
- Deployed using GitHub Pages

### Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- CSS
- GitHub Pages

This project is part of the IBM / CognitiveClass React and Redux final project.
