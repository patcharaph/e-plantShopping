# e-plantShopping

e-plantShopping is a React-based shopping application for purchasing house plants.
Users can browse different categories of plants, add items to a shopping cart,
adjust quantities, and view the total cost before checkout.

This project demonstrates the use of:
- React functional components
- Redux Toolkit for application state management
- useSelector and useDispatch for interacting with the Redux store
- Dynamic UI updates based on user actions

The application includes:
- A landing page
- A product listing page
- A shopping cart page

This project was developed as part of a React final project.
